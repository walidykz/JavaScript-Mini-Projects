### 2. Number Guessing Game 🎯

A number guessing game where the player tries to find a randomly generated number based on the selected difficulty level.

**Features:**
- Three difficulty levels:
  - Easy: Numbers between 1 and 50 (10 attempts)
  - Medium: Numbers between 1 and 100 (15 attempts)
  - Hard: Numbers between 1 and 500 (20 attempts)
- Random number generation
- Attempt counter
- Input validation
- "Too High" and "Too Low" hints
- Game over system
- Restart functionality
- Interactive user interface

**Technologies:**
- HTML
- CSS
- JavaScript
